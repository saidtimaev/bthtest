# Система управления товарами (Тестовое задание)

Тестовое задание: CRUD для управления товарами на Laravel + Inertia.js.

## Стек технологий
* **Backend:** PHP 8.x, Laravel 13
* **Frontend:** Vue 3 (Composition API), TypeScript, Tailwind CSS
* **State/Routing:** Inertia.js
* **Database:** PostgreSQL

## Основные фичи
* Авторизация по токену (SPA).
* Управление товарами (CRUD).
* Валидация на стороне сервера (FormRequests).
